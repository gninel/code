# 教学设计默认值

## 问题链
核心问题：如何准确确定物体的位置？（兼顾确定位置的三要素、观测点变化与生活应用）
一、吸引环节：激活经验，发现矛盾（对应目标3，铺垫目标1）
1、生活情境提问：“周末全家去动物园，妈妈说‘熊猫馆在大门的东边’，你能直接找到吗？为什么？”
二、探索环节：自主建构，理解核心（对应目标1、2）
2、操作提问：“以动物园大门为观测点，熊猫馆在大门的东北方向 —— 如果用量角器测量，从正北方向向东转 30°，这种方向可以怎么描述？（引导说出 “北偏东 30°”）”
3、补充提问：“知道了熊猫馆在大门的北偏东30°方向，就能确定位置了吗？如果熊猫馆距离大门 100米，而鸟馆在大门北偏东 30°方向 50 米处，它们的位置一样吗？”
4、小组合作：“要唯一确定一个物体的位置，必须包含哪三个关键信息？缺少其中一个会有什么问题？”
5、情境转换：“从大门出发，走到熊猫馆后，现在以熊猫馆为观测点，大门在熊猫馆的什么方向？和之前‘熊猫馆在大门的北偏东 30°’相比，方向有什么变化？”
三、解释环节：梳理方法，规范表达（对应目标 1、2）
6、归纳提问：“请用自己的话总结：怎样才能准确描述物体的位置？（引导学生说出 “先确定观测点→再看方向（北偏东 / 西、南偏东 / 西 + 角度）→最后说距离”）”
7、原理追问：“为什么确定位置必须以‘观测点’为基准？如果观测点不明确，会出现什么情况？”
四、深化环节：迁移应用，拓展思维（对应目标 1、2、3）
8、生活应用：“你家在学校的什么位置？请用‘方向 + 角度 + 距离’的方式描述，再反过来描述学校在你家的位置。”
9、创新设计：“如果让你设计一个‘校园寻宝图’，在图上标注 3 个宝藏地点，用方向、角度和距离描述每个宝藏相对于起点的位置，让同桌根据你的描述寻找宝藏。”
10、反思提问：“通过本节课的学习，你学会了什么？在确定位置时，你最容易出错的地方是什么？生活中还有哪些地方会用到‘方向 + 角度 + 距离’确定位置？”

## 活动设计
一、游戏情境：创设寻宝任务：
"学校少先队要举办校园定向活动，需要设计藏宝图，用最准确的线索帮助同伴寻宝"

二、校园寻宝设计活动流程（10分钟）
1.设计任务（4分钟）
每组在校园平面图上标注2个"宝藏点"
制作寻宝线索卡，要求：
	每个点用"方向+角度+距离"描述
	可在1个宝藏点设置一个干扰项（如缺少距离信息）
2.寻宝挑战（4分钟）
交换线索卡，根据描述寻找对方宝藏
记录寻宝过程中的困惑点
3.反思改进（2分钟）
分析干扰项导致的寻宝失败案例
优化线索卡，补充完整三要素

三、活动设计亮点说明：
1.情境闭环：
课堂引入（家校位置）→游戏应用（寻宝图）形成认知闭环
2.错误分析机制
故意设置的干扰项促使学生自主发现三要素必要性，寻宝失败案例成为最佳教学资源
3．核心素养落实：
空间观念：通过实地场景与平面图的多次转换强化
应用意识：解决真实的位置描述问题
创新意识：自主设计寻宝线索

***小组合作学习活动设计示例
《确定位置》中的问题链输入：
**动手探究：“结合教材情境图（比例尺 1:10000，图上 1 厘米代表实际 100 米），量一量：猴山在大门的什么方向？图上距离是多少？实际距离是多少？”
**小组合作：“要唯一确定一个物体的位置，必须包含哪三个关键信息？缺少其中一个会有什么问题？”
**教学活动设计示例：
一、活动任务
1.	以小组为单位，结合教材情境图（比例尺 1:10000，图上 1 厘米代表实际 100 米），测量并记录 “猴山相对于大门” 的方向、图上距离，计算实际距离。
2.	讨论并总结：要唯一确定一个物体的位置，必须包含哪三个关键信息？分别分析缺少 “方向”“角度”“距离” 中任一信息时，会出现什么问题。
3.	完成活动记录单，小组代表准备汇报成果。
二、活动流程（总时长：20 分钟）
1. 个人思考（5 分钟）
•	每位学生独立观察教材情境图，明确 “大门” 为观测点，尝试用自己的方法找到猴山的位置。
•	自主思考：如何测量方向（北偏东 / 西等）？需要用到什么工具？图上距离怎么量？实际距离如何根据比例尺计算？
•	初步记录自己的测量思路和预估数据（不要求精准，侧重独立思考）。
2. 组内交流（5分钟）
•	测量分享：测量员展示各自的测量方法和数据，组内讨论 “为什么测量结果可能不同”（如量角器摆放是否标准、直尺测量是否对齐刻度），统一测量标准（量角器中心对准观测点 “大门”，0 刻度线与正北 / 正南方向重合；直尺测量时从大门中心点到猴山中心点）。
•	计算交流：记录员分享实际距离的计算过程（图上距离 ×100 米），组内核对计算结果，确保无误。
•	问题讨论：围绕 “确定物体位置需要哪些信息” 展开交流，结合 “只说方向不说距离”“只说距离不说方向”“说方向但不说角度” 的具体例子，分析每种情况能否唯一确定位置（如只说 “猴山在大门的北偏东方向”，无法确定是猴山还是其他同方向的景点；只说 “猴山距离大门 200 米”，无法确定具体方向）。
3. 小组汇总（2分钟）
•	组长牵头，整合组内测量的最终数据（方向、图上距离、实际距离），填写在活动记录单对应位置。
•	共同提炼确定物体位置的三个关键信息，梳理缺少每个信息的具体问题，形成小组统一结论。
4. 组间互评（3 分钟）
•	每个小组派发言人进行 1 分钟成果汇报，分享测量结果和结论。
•	其他小组针对汇报内容提问或补充（如 “你们测量角度时，是从正北还是正东开始量的？”“如果比例尺变了，实际距离计算方法会变吗？”），汇报小组进行回应。
•	互评聚焦：测量数据的准确性、结论的完整性、表达的清晰度。
