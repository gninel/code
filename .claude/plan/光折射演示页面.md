# 光折射演示页面开发计划

## 任务上下文
**任务描述**: 生成一个用于中学物理课堂教学上在线演示的页面，内容为光的折射理解。

**目标用户**: 八年级学生和物理教师

**核心需求**:
- 理解折射定律、折射率概念
- 课堂讲解辅助工具
- 可交互演示（角度调节、介质选择、路径变化）
- 符合人教版教学大纲

## 技术方案
**选择方案**: Canvas 2D物理模拟演示
**技术栈**: HTML5 Canvas + 原生JavaScript + CSS3
**架构**: 单页面应用（SPA）

## 实施计划

### 阶段1: 项目结构搭建
- 创建单页面HTML文件
- 设计Canvas渲染区域
- 搭建CSS基础框架

### 阶段2: 核心物理引擎
- 实现斯涅尔定律计算
- 创建光线追踪算法
- 建立介质折射率数据库

### 阶段3: 交互控制系统
- 入射角度滑块(0-90度)
- 介质选择器
- 实时参数显示

### 阶段4: 可视化渲染
- Canvas光线路径绘制
- 界面法线标记
- 动态效果

### 阶段5: 教学内容集成
- 折射定律原理说明
- 实时公式展示
- 教学要点

### 阶段6: 教师辅助功能
- 演示模式
- 重置功能
- 预设场景

## 核心组件设计
- RefractionEngine - 物理计算引擎
- LightRay - 光线对象
- Medium - 介质管理
- InteractionController - 交互控制
- Renderer - Canvas渲染器

## 物理公式
- 折射定律: n₁ × sin(θ₁) = n₂ × sin(θ₂)
- 临界角: θc = arcsin(n₂/n₁)

## 预设介质
- 空气: n = 1.00
- 水: n = 1.33
- 玻璃: n = 1.50
- 钻石: n = 2.42
