# 物理实验生成系统开发计划

## 任务概述
利用LLM+开源PHET来自动生成物理实验 - 独立系统开发

## 技术方案选择
**方案2：基于PHET HTML模板的定制方案**
- 用户自然语言输入 → Kimi LLM分析 → 模板匹配 → 参数配置 → 生成定制HTML
- 轻量级、快速开发、易维护

## 项目结构
```
physics-experiment-generator/
├── index.html                 # 主对话界面
├── assets/
│   ├── css/style.css         # 主样式文件
│   ├── js/                   # 核心JavaScript文件
│   └── templates/            # PHET实验模板库（23个）
├── config/                   # 配置文件
├── generated/               # 生成的实验文件
└── README.md               # 使用说明
```
