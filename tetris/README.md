# 🎮 移动端俄罗斯方块游戏

一个功能完整的HTML5俄罗斯方块游戏，专为移动设备优化，支持触控操作和难度调节。

## ✨ 特性

- 📱 **移动端优化**: 完美适配手机屏幕，支持触控操作
- 🎯 **难度控制**: 4个难度等级（简单、普通、困难、地狱）
- 🏆 **最高分记录**: 使用localStorage自动保存最高分
- 🎨 **现代UI设计**: 渐变色彩、圆角设计、流畅动画
- ⌨️ **双重控制**: 同时支持键盘和触控操作
- 👻 **幽灵方块**: 实时显示落点预览
- 📊 **完整统计**: 分数、等级、消除行数实时显示
- 🔄 **自动升级**: 每消除10行自动提升等级，速度加快

## 🎮 游戏操作

### 移动端（触控）
- **← →**: 左右移动按钮
- **↓**: 加速下落
- **↻**: 旋转方块
- **⬇**: 硬降（直接落到底部）

### 桌面端（键盘）
- **← →**: 方向键左右移动
- **↑**: 方向键上旋转
- **↓**: 方向键下加速
- **空格**: 硬降

## 🚀 快速开始

### 方式1: 直接在浏览器中打开
```bash
# 使用Python简易服务器
cd tetris
python3 -m http.server 8000

# 或使用Node.js
npx serve .
```

访问 `http://localhost:8000` 即可开始游戏。

### 方式2: 双击index.html
直接在浏览器中打开 `index.html` 文件即可运行。

## 📋 游戏规则

1. **基础玩法**: 7种不同形状的方块从顶部下落，玩家需要控制它们的位置和旋转，使其在底部堆叠
2. **消除行**: 当一行被完全填满时，该行会被消除，上方的方块会下落
3. **得分规则**:
   - 消除1行: 100分 × 等级
   - 消除2行: 300分 × 等级
   - 消除3行: 500分 × 等级
   - 消除4行: 800分 × 等级
   - 软降: 每格1分
   - 硬降: 每格2分
4. **升级**: 每消除10行提升1级，方块下落速度加快
5. **游戏结束**: 当新方块无法放置在游戏区域时，游戏结束

## 🎚️ 难度设置

| 难度 | 速度倍率 | 描述 |
|------|---------|------|
| 简单 | 1.0× | 适合初学者，下落速度较慢 |
| 普通 | 0.8× | 标准速度，平衡的挑战 |
| 困难 | 0.6× | 快速下落，需要快速反应 |
| 地狱 | 0.4× | 极速模式，高手挑战 |

**注意**: 难度需要在游戏开始前选择，游戏中无法更改。

## 🛠️ 技术栈

- **HTML5**: 游戏结构

- **JavaScript (ES6+)**: 游戏逻辑、Canvas绘制

- **LocalStorage API**: 最高分存储

## 🎨 游戏特色

### 1. SRS旋转系统
采用标准的SRS（Super Rotation System）旋转系统，支持墙壁踢（Wall Kick），让旋转更加流畅自然。

### 2. 幽灵方块
半透明显示方块的最终落点，帮助玩家更好地规划位置。

### 3. 渐进式难度
随着等级提升，下落速度逐渐加快，每级速度降低10%。

### 4. 精美视觉效果
- 渐变色方块
- 高光效果
- 流畅动画
- 现代化UI

## 🎯 核心功能代码

### 游戏配置
- 游戏区域: 10列 × 20行
- 方块大小: 30px × 30px
- 7种经典俄罗斯方块形状(I, O, T, S, Z, J, L)

### 主要类和方法
- `Game`: 游戏主类
- `collision()`: 碰撞检测
- `rotate()`: 方块旋转（含墙壁踢）
- `clearLines()`: 行消除逻辑
- `updateSpeed()`: 动态速度调整
- `saveHighScore()`: 最高分持久化
